# Code of Conduct

为了让这个仓库保持可协作、可讨论、可持续，参与者默认同意以下原则：

- 尊重彼此，讨论问题时聚焦事实和改进方案
- 接受不同经验水平的贡献者，避免嘲讽、羞辱或人身攻击
- 对 bug、误解和回归保持建设性态度
- 在公开讨论中避免泄露安全细节、隐私数据或第三方敏感信息

以下行为不受欢迎：

- 骚扰、歧视、威胁或持续性敌意表达
- 明知无关仍反复刷屏、引战或破坏讨论
- 未经允许公开他人的敏感信息

维护者可以编辑、隐藏、锁定或移除不合适的内容，并在必要时限制参与权限。

如果你发现不当行为，可以通过仓库 issue 先做简要说明，维护者会尽量跟进处理。
